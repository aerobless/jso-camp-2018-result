JSO Cloud Functions API Usage:

  • GET /api/kickerbox returns all kickerboxes
  • GET /api/kickerbox/:id returns a specific kickerbox
  • GET /api/result returns all results
  • GET /api/result/:id returns a specific result
  • POST /api/result/ post a new result
  • GET /api/challenge returns all challenges
  • GET /api/challenge/:id returns a specific challenge
  • POST /api/challenge/ post a new challenge
  • PUT /api/challenge/:id update a specific challenge
